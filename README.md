## Теги в HTML

Структура парного тега:
**<тег>тело< /тег>**
*Пример:*
```html
<h1>Это заголовок</h1>
```
## Базовые HTML-теги
Примеры:
```html
<h2>Заголовок</h2>
<p>Абзац текста</p>
<strong>Жирный</strong>
<em>Курсив</em>
<hr>
<a href="https://example.com">Ссылка</a>
<img src="example.jpg" alt="Описание">
```
## Парные и одиночные теги
Одиночный тег
Не имеет закрывающей части.
Пример:
```html
<img src="https://google.com">
```
Парный тег
Состоит из открывающего и закрывающего тегов
Пример:
```html
<h1>Это заголовок<h2>
```
## Списки в HTML
Нумерованный список:
```html
<ol>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
    <li>Третий пункт</li>
</ol>
```
Маркированый список:
```html
<ul>
    <li>Элемент списка</li>
    <li>Ещё один элемент</li>
</ul>
```
Вложенные списки:
```html
<ul>
    <li>Пункт 1
        <ol>
            <li>Вложенный 1</li>
            <li>Вложенный 2</li>
        </ol>
    </li>
</ul>
```
## Атрибуты
Атрибуты — это дополнительные параметры тега. Они уточняют, как именно
должен работать элемент: задают ссылку, размер, стиль, язык, поведение и многое
другое.
Пример
```html
<тег атрибут="значение">..</тег>
```
